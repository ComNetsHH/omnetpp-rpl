\begin{DoxyParams}{Parameters}
{\em dio} & D\+IO packet received recently \\
\hline
\end{DoxyParams}
Maintain following sets of link-\/local nodes\+:
\begin{DoxyItemize}
\item backup parents
\item candidate parents where preferred parent is chosen from the candidate parent set \mbox{[}R\+F\+C6560, 8.\+2.\+1\mbox{]}
\end{DoxyItemize}

In current implementation, neighbor data is represented by most recent D\+IO packet received from it.

If D\+IO sender has equal rank, consider this node as a backup parent
